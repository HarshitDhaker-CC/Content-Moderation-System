About the Model

Built on State-of-the-Art ML

ToxiGuard AI is built on a NLP and Random Forest classifier model, trained on the Kaggle Toxic Comment Classification Challenge dataset containing over 159,000 labeled Wikipedia comments.

The system employs a multi label classification approach, capable of detecting multiple simultaneous toxicity categories within a single piece of text.

Dataset Kaggle Toxic Comment Classification (159K+ samples)
Base Model Random Forest and NLP with fine tuned classification head
Performance ROC-AUC: 0.987 | F1-Score: 0.94 | Precision: 0.96
Backend Python, Flask, HuggingFace Transformers, scikit-learn
